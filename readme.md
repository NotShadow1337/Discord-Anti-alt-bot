# Installation
```
pip install pycord
```
# config.json
```json
{
    "token":"bot.token",
    "guild_id":"guild.id",
    "emojis":{
        "success":":white_check_mark:",
        "failure":":x:"
    }
}
```


# Features

- Slash commands
- Member join logs and alt logs
- Auto kick alts
- Customisable requirements
- Custom logs channel
